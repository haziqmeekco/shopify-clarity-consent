# Shopify Clarity Consent

Cookie consent integration for Microsoft Clarity analytics in Shopify themes.

## CDN URL

```
https://haziqmeekco.github.io/shopify-clarity-consent/clarity-consent.js
```

## Usage

Add this line to your `layout/theme.liquid` file, before the closing `</body>` tag:

```liquid
<script src="https://haziqmeekco.github.io/shopify-clarity-consent/clarity-consent.js" defer="defer"></script>
```

## What It Does

Automatically syncs Shopify's customer privacy consent preferences with Microsoft Clarity analytics.

## Version

Current version: 1.0.0

## Updates

To update all stores, simply push changes to this repository. All stores will automatically use the updated version.
